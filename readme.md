# **Project Consol**

#### **Setup steps**

###### **APP**
`cd app`<br/>
`npm install`<br/>
`npm run serve`
###### **Cloud Functions**
`npm install -g firebase-tools`<br/>
`cd functions`<br/>
`npm install`<br/>
`firebase emulators:start --only functions`<br/>
